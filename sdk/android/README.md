# AEAMC Android SDK

This directory contains the Android SDK for the Adaptive Energy-Aware Mobile Consensus (AEAMC) project.

## Purpose

The purpose of this SDK is to provide a lightweight library for Android devices to:
- Collect and report device telemetry.
- Execute assigned tasks.
- Generate Proof-of-Task (PoT) and Proof-of-Availability (PoA).
- Communicate with the microblock coordinator.

## Structure

- `app`: The main application module.
- `app/src/main/java`: The root of the Kotlin source code.
- `app/src/main/res`: Android resources (layouts, strings, etc.).
- `build.gradle`: The top-level Gradle build script.
- `app/build.gradle`: The application-level Gradle build script.
